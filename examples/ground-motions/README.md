# Ground Motion Sample Data

H24道路橋示方書の地震波データサンプル集。

## 概要

時刻歴応答解析の地震波読込検証に使用するサンプルデータ。

## ファイル構成

```
h24/
├── level2-type2.csv   # H24道路橋示方書 Level2 Type2 波形（CSV形式）
└── level2-type2.xls   # 同上（Excel形式）
```

## H24道路橋示方書

道路橋示方書・同解説（平成24年版）に定められた地震波データ。

- 9波形: Ⅱ-Ⅰ-１, Ⅱ-Ⅰ-２, Ⅱ-Ⅰ-３, Ⅱ-Ⅱ-１, Ⅱ-Ⅱ-２, Ⅱ-Ⅱ-３, Ⅱ-Ⅲ-１, Ⅱ-Ⅲ-２, Ⅱ-Ⅲ-３
- 単位: gal (cm/s²)
- 時間刻み: 0.01秒
- 先頭列: 時間（秒）

## 使い方

1. 時刻歴応答解析を開く
2. 地震波設定を開く
3. H24道示波形インポートをクリック
4. CSVファイルを選択
5. 検出された波形から使用するものを選択
6. X / Y / Z 方向へ割り当て
