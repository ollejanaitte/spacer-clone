### 有効質量

```text

```

絶対有効質量を保持する。

---

### 有効質量比

```text
effectiveMassRatios
```

各モード単独の有効質量比を保持する。

---

### 累積参加率

```text
cumulativeEffectiveMassRatios
```

モード順累積値を保持する。

累積値は強制的に 1.0 に丸めない。

