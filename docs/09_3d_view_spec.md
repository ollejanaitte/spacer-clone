# 09 3D View Specification

## 1. 目的

Three.jsを用いたMVPの3D線モデル表示仕様を定義する。解析入力と結果を視覚確認できることを目的とし、CAD編集機能は持たせない。

## 2. 対象範囲

- 節点表示。
- 部材線表示。
- 支点記号。
- 荷重矢印。
- 部材番号・節点番号ラベル。
- 変形図。
- 表示倍率。
- カメラ操作。
- 選択ハイライト。

## 3. 非対象範囲

- 断面形状の3Dソリッド表示。
- CAD編集、スナップ、ドラッグ作成。
- DXF表示・出力。
- 影響線図、モード図、応答スペクトル結果図。
- 温度荷重やプレストレス専用表示。
- 高度な帳票図面作成。

## 4. 表示仕様

### 入力データ

- `project.json` 互換モデル。
- 選択中荷重ケースID。
- 任意の解析結果JSON。
- UIの選択状態。
- 表示設定。

### 節点表示

- 各節点を点または小球で表示する。
- 選択節点はハイライト色にする。
- 支点がある節点は支点記号を重ねる。

### 部材線表示

- `nodeI` から `nodeJ` へ直線を描画する。
- 選択部材は色と線幅を変える。
- 部材IDラベル表示を切り替え可能にする。

### 支点記号

- 拘束節点に簡易記号を表示する。
- 完全固定はブロックまたは三角記号でよい。
- 部分拘束の詳細DOFはプロパティパネルで確認する。

### 荷重矢印

- 選択中荷重ケースの節点集中荷重を矢印表示する。
- 部材等分布荷重を複数矢印または帯状記号で表示する。
- 相対値が分かるように自動スケールする。
- 力の向きは必ず実ベクトル方向と一致させる。

### ラベル

- 節点番号ラベルON/OFF。
- 部材番号ラベルON/OFF。
- ラベルはカメラ方向を向く。
- モデルが密な場合はラベル非表示にできる。

### 変形図

- 解析結果がある場合、選択荷重ケースの変位を用いて変形後形状を表示する。
- MVPでは部材を変形後節点間の直線で描いてよい。
- 変形前形状は薄色で残す。
- 変形倍率は手動または自動。

### 表示倍率

設定:

- `deformationScale`
- `loadScale`
- `nodeSize`
- `labelSize`

自動変形倍率:

1. モデルのバウンディングボックス対角長を計算する。
2. 最大並進変位を計算する。
3. 最大変位が対角長の一定割合になるよう倍率を決める。
4. 最大変位が0の場合、変形図を表示しない。

### カメラ操作

- Orbit。
- Pan。
- Zoom。
- Fit to model。
- XY、XZ、YZ、Isometric。

### 選択ハイライト

- 節点クリックで節点選択。
- 部材クリックで部材選択。
- 選択状態はReactへ通知する。
- React側の選択変更も3Dビューへ反映する。

## 5. エラー処理

- 不正参照部材があってもビュー全体をクラッシュさせない。
- 描画不能エンティティは警告としてUIへ返す。
- 結果JSONがない場合、変形図を非表示にする。
- 変位に非有限値がある場合、変形図を表示せずエラー表示する。

## 6. テスト観点

- 空モデルでクラッシュしない。
- 節点・部材を表示できる。
- 支点、荷重矢印、ラベルON/OFFが動作する。
- 選択ハイライトがReact状態と同期する。
- 変形図が結果JSONから描画される。
- 1,500節点、2,500部材程度で操作可能な設計になっている。

## 7. 完了条件

- MVPモデルを線表示できる。
- 支点と荷重の概略確認ができる。
- 解析結果の変形図を表示できる。
- UI担当がコンポーネントとして組み込める。
- MVP外の高度図面機能を含まない。
