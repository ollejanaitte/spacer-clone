# History

**Authority:** HISTORICAL / NAVIGATION
**Status:** ACTIVE (index only)
**Note:** Delivery evidence is not current fact and not target authority.

This index labels retained evidence. The listed frame, application, and road historical clusters are stored under `docs/history`. Redirect stubs remain for compatibility where frozen or root-inbound references still use old paths. Other historical material that has not been relocated remains at its current path.

## How to Use Historical Documents

- Treat phase logs, handovers, investigations, PR notes, and release notes as **evidence**.
- Do not promote historical TODOs or unaccepted proposals into requirements without a Stage 6–10 decision.
- Prefer CURRENT FACT ([../scoping/README.md](../scoping/README.md)) and TARGET DECISION ([../planning/stage6-10/README.md](../planning/stage6-10/README.md)) for ownership and contracts.
- FUTURE PROPOSAL material under untracked `docs/bridge-modeler-v2/**` is excluded and must not be indexed as authority.
