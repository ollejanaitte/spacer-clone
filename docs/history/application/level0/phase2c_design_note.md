# Phase 2C 設計メモ: 結果翻訳ルール

## 設計日時

2026-06-21

## 目的

解析結果を初心者向けに翻訳するルールの設計

## 1. 結果翻訳ルール案

### 翻訳原則

1. **専門用語を避ける**: 「固有周期」→「橋が揺れやすい時間」
2. **比較で説明する**: 「0.85秒」→「人間が1歩歩くより少し短い時間」
3. **具体例を出す**: 「変位0.025m」→「約2.5cm動く」
4. **影響を説明する**: 「変位が大きい」→「橋の揺れが大きい」

### 翻訳ルール表

| 専門用語 | 翻訳 | 説明 |
|----------|------|------|
| 固有周期 | 橋が揺れやすい時間 | 橋が自然に揺れる周期 |
| 最大変位 | 最も動いた量 | 橋がどれだけ動いたか |
| 反力 | 支点にかかる力 | 橋を支える部分にかかる力 |
| 断面力 | 橋にかかる力 | 橋の部材に作用する力 |
| 有効質量比 | 揺れに使うmassの割合 | 地震時に揺れるmassの割合 |

## 2. 閾値案

### 変位の閾値

| レベル | 閾値 | 表示メッセージ |
|--------|------|----------------|
| 小 | < 0.005m | 「橋の揺れはほとんどありません」 |
| 中 | 0.005m〜0.02m | 「橋が少し揺れます」 |
| 大 | > 0.02m | 「橋が大きく揺れます」 |

### 固有周期の閾値

| レベル | 閾値 | 表示メッセージ |
|--------|------|----------------|
| 短い | < 0.5s | 「橋がしっかりしています」 |
| 普通 | 0.5s〜1.5s | 「一般的な橋です」 |
| 長い | > 1.5s | 「橋が柔らかいです」 |

### 反力の閾値

| レベル | 閾値 | 表示メッセージ |
|--------|------|----------------|
| 小 | < 100kN | 「支点にかかる力は小さいです」 |
| 中 | 100kN〜500kN | 「支点にかかる力は普通です」 |
| 大 | > 500kN | 「支点にかかる力は大きいです」 |

## 4. 実装方針

### データ構造

```typescript
type TranslatedResult = {
  label: string;           // 専門用語
  value: string;           // 数値
  translatedValue: string; // 翻訳された値
  description: string;     // 説明
  level: "safe" | "caution" | "danger"; // レベル
};
```

### 翻訳関数

```typescript
function translateResult(result: AnalysisResult): TranslatedResult[] {
  return [
    {
      label: "最大変位",
      value: `${maxDisplacement}m`,
      translatedValue: `約${maxDisplacement * 100}cm動きます`,
      description: getDisplacementDescription(maxDisplacement),
      level: getDisplacementLevel(maxDisplacement),
    },
    // ...
  ];
}
```

### UIコンポーネント

- `ResultTranslationCard`: 翻訳結果を表示するカード
- `TranslationLevelBadge`: レベルを表示するバッジ
- `TranslationDescription`: 説明テキスト

## 5. 想定リスク

| リスク | 影響度 | 対策 |
|--------|--------|------|
| 翻訳の不正確さ | 中 | 専門家レビュー |
| 閾値の適切さ | 中 | ユーザーテスト |
| 多言語対応 | 低 | i18n対応 |
| パフォーマンス | 低 | 遅延計算 |
