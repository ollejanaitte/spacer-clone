## 概要

SPACER Clone（本ソフト）の解析結果を、SPACER実機出力および理論値と比較検証する計画。

## 比較対象モデル一覧

### 1. 片持ち梁（Cantilever Beam）

- **説明**: 一端固定・他端自由の梁。先端集中荷重で最大変位・反力・モーメントを確認。
- **入力**: 固定支点1個、梁1本、先端集中荷重
- **期待変位**: δ = PL³ / (3EI)
- **期待反力**: 固定端でP, PL
- **期待断面力**: Mmax = PL
- **許容誤差**: 1%以内
- **比較観点**: 基本的な梁理論との一致確認

### 2. 単純梁（Simply Supported Beam）

- **説明**: 両端ピン支点の梁。中央集中荷重で変位・反力・モーメントを確認。
- **入力**: ピン支点2個、梁1本、中央集中荷重
- **期待変位**: δ = PL³ / (48EI)
- **期待反力**: 両端でP/2
- **期待断面力**: Mmax = PL/4
- **許容誤差**: 1%以内
- **比較覦点**: 基本梁理論との一致確認

### 3. 門型ラーメン（Portal Frame）

- **説明**: 2柱・1梁の門型フレーム。梁中央に水平荷重。
- **入力**: 柱2本、梁1本、固定支点2個、水平荷重
- **期待値**: 柱上端モーメント、梁端モーメント、反力
- **許容誤差**: 5%以内
- **比較観点**: 骨組解析の基本動作確認

### 4. 3D小規模骨組

- **説明**: 3次元の小規模骨組モデル。複数方向荷重。
- **入力**: 複数節点・部材、3方向荷重
- **期待値**: 各方向の変位・反力・断面力
- **許容誤差**: 5%以内
- **比較観点**: 3次元解析の正確性確認

### 5. 支点反力確認モデル

- **説明**: 各種支点条件（ピン、ローラー、固定）での反力確認。
- **入力**: 異なる支点条件を持つ梁
- **期待値**: 支点反力（各方向・各モーメント）
- **許容誤差**: 1%以内
- **比較観点**: 支点条件の正しい実装確認

### 6. 断面力確認モデル

- **説明**: 等分布荷重・集中荷重による断面力分布の確認。
- **入力**: 等分布荷重、集中荷重
- **期待値**: せん断力図、曲げモーメント図
- **許容誤差**: 1%以内
- **比較観点**: 断面力計算の正確性確認

## 検証手順

1. 各モデルの入力JSONを作成
2. SPACER Cloneで解析実行
3. 解析結果（変位、反力、断面力）をCSV出力
4. 理論値またはSPACER実機出力と比較
5. 誤差を計算し許容範囲内か確認
6. 結果を `spacer-comparison-results.md` に記録

## 期待値の考え方

- 理論値は構造力学の基本公式から計算
- 簡単なモデル（片持ち梁、単純梁）は理論値との比較が可能
- 複雑なモデル（門型ラーメン、3D骨組）はSPACER実機出力との比較が理想
- SPACER実機出力が手元にない場合は、理論値および既知の数値解との比較に留める

## 許容誤差

| モデル | 許容誤差 | 理由 |
|--------|----------|------|
| 片持ち梁 | 1%以内 | 基本梁理論との直接比較が可能 |
| 単純梁 | 1%以内 | 基本梁理論との直接比較が可能 |
| 門型ラーメン | 5%以内 | 骨組解析の数値誤差を考慮 |
| 3D小規模骨組 | 5%以内 | 3次元効果を含む |
| 支点反力 | 1%以内 | 支点条件の正確性確認 |
| 断面力 | 1%以内 | 分布荷重の積分精度確認 |

## 未検証項目

- SPACER実機との直接比較（実機出力が手元にない場合）
- 非線形解析
- 大変位解析
- 時刻歴応答解析の詳細検証
- 移動荷重解析の詳細検証

## SPACER実機照合方法

SPACER実機出力を取り込む手順:

1. SPACERで解析実行し、結果ファイルをエクスポート
2. エクスポートされたCSV/JSONを本ソフトの入力形式に変換
3. 本ソフトで同じモデルを解析
4. 結果を比較
5. 差異がある場合は原因を調査

**注意**: SPACER実機出力が手元にない場合は、比較準備完了状態とする。
