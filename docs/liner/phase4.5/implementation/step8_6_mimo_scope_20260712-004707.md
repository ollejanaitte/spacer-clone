## 9. I/J反転時の local axis 変化

**事実** (`element.py:10-37`)
- I/J反転 → `x_axis` が符号反転（-x_axis）
- `z_axis = cross(-x_axis, y_axis) = -cross(x_axis, y_axis)` → z_axis も符号反転

- **結果**: x_axis, z_axis が反転、y_axis は同一。ローカル→グローバル変換の Rotation 行列の x 行, z 行が反転。

**推定**: `element.py:25-36` の計算から導出。座標系の定義により結果は明確。
